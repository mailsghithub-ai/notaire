# Projet Notaires de France - Espace Sécurisé

## Installation

1. Clonez le repository
2. Installez les dépendances :
```bash
npm install
```

3. Configurez le fichier `.env` avec vos informations

4. Lancez le projet :
```bash
npm start
```

## Build pour production

```bash
npm run build
```

## Sécurité

- Les informations sensibles sont stockées dans le fichier `.env`
- Ne jamais commiter le fichier `.env`
- Utiliser `.env.local` pour le développement local

## Technologies utilisées

- React 18
- CSS3 avec animations
- API Discord pour les notifications
